# ショートステイ利用記録

GitHub Pages に置く最低限の構成です。

## 置くファイル
- index.html
- install.html
- apple-touch-icon.png
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- icon-192.png
- icon-512.png
- site.webmanifest

## 補足
- iPhone / iPad のホーム画面追加用に apple-touch-icon.png を使います。
- install.html は追加手順用のページです。
- 本体 URL は HTML 内の EXEC_URL を必要に応じて差し替えてください。
